## 强化学习

### Q 学习
好，那我们完全不用 gym，自己手写一个最简单的迷宫游戏，只用 numpy + 随机，代码超短，一眼看懂强化学习在干嘛。
场景：
- 一共 5 个格子： [0,1,2,3,4] 
- 起点在 0，目标是走到 4
- 只能向左 / 向右走
- 走到 4 奖励 +10，游戏结束
- 其他步奖励 0
用最简单的 Q-Learning 来学。
纯手写极简强化学习代码（可直接运行）

```python
import numpy as np
import random

# ===================== 环境设定 =====================
n_states = 5  # 5个格子：0 1 2 3 4
n_actions = 2  # 0=左，1=右
goal_state = 4

# ===================== Q表初始化 =====================
Q = np.zeros((n_states, n_actions))

# ===================== 超参数 =====================
lr = 0.1  # 学习率
gamma = 0.9  # 未来奖励折扣
epsilon = 0.1  # 探索概率
episodes = 1000

# ===================== 训练 =====================
for episode in range(episodes):
    s = 0  # 每次从起点开始
    done = False
    while not done:
        # 探索 or 利用
        if random.uniform(0, 1) < epsilon:
            a = random.choice([0, 1])  # 随机走
        else:
            a = np.argmax(Q[s])  # 走Q值最大的

        # 执行动作
        if a == 1:  # 右
            s_next = s + 1
        else:  # 左
            s_next = s - 1 if s > 0 else 0

        # 奖励
        if s_next == goal_state:
            r = 10
            done = True
        else:
            r = 0

        # Q-Learning 更新公式
        Q[s, a] = Q[s, a] + lr * (r + gamma * np.max(Q[s_next]) - Q[s, a])
        s = s_next

# ===================== 训练完成，看结果 =====================
print("训练完成！Q 表：")
print(Q)
print("\n走一遍最优路径：")
s = 0
path = [s]
while s != goal_state:
    a = np.argmax(Q[s])
    if a == 1:
        s += 1
    else:
        s = max(0, s - 1)
    path.append(s)
print("路径:", path)
```

```plaintext
[ 0. 9. ]
[ 0. 10. ]
[ 0. 10. ]
[ 0. 0. ]]
走一遍最优路径：
路径: [0, 1, 2, 3, 4]
```

核心一句话讲懂
- Q 表：记录每个位置“向左/向右”有多好
- 智能体一开始乱走
- 走到终点给奖励，反向更新 Q 值
- 最后学会：一直向右走

### 策略学习
好的！下面我用 **策略梯度（Policy Gradient）** 来重写这个极简迷宫游戏，核心思想是：**直接学习一个策略函数 $\pi(a|s)$，而不是学 Q 值**。

---

 策略梯度（REINFORCE）实现

```python
import numpy as np
import random

# ===================== 环境设定 =====================
n_states = 5
n_actions = 2  # 0=左，1=右
goal_state = 4

# ===================== 策略参数初始化 =====================
# 每个状态有2个动作的概率权重（softmax 后得到概率）
theta = np.zeros((n_states, n_actions))  # 策略参数

# ===================== 超参数 =====================
lr = 0.01          # 学习率
gamma = 0.9        # 折扣因子
episodes = 2000    # 训练轮数

# ===================== softmax 策略 =====================
def policy(s):
    """返回状态 s 下各动作的概率"""
    exp = np.exp(theta[s])
    return exp / np.sum(exp)

# ===================== 训练 =====================
for episode in range(episodes):
    # 存储一条轨迹
    states = []
    actions = []
    rewards = []

    s = 0
    done = False
    while not done:
        # 根据策略采样动作
        probs = policy(s)
        a = np.random.choice([0, 1], p=probs)

        # 执行动作
        if a == 1:  # 右
            s_next = s + 1
        else:       # 左
            s_next = s - 1 if s > 0 else 0

        # 奖励
        if s_next == goal_state:
            r = 10
            done = True
        else:
            r = 0

        # 记录
        states.append(s)
        actions.append(a)
        rewards.append(r)

        s = s_next

    # ===== 计算折扣回报 =====
    G = 0
    returns = []
    for r in reversed(rewards):
        G = r + gamma * G
        returns.insert(0, G)

    # ===== 策略梯度更新 =====
    for t in range(len(states)):
        s = states[t]
        a = actions[t]
        G_t = returns[t]

        # softmax 策略的梯度：∇log π(a|s) = 1 - π(a|s) 对于选中动作，-π(a'|s) 对于其他动作
        probs = policy(s)
        grad = -probs.copy()          # 所有动作的负概率
        grad[a] += 1                  # 选中动作 +1

        # 更新参数：θ += lr * G_t * ∇log π(a|s)
        theta[s] += lr * G_t * grad

# ===================== 训练完成，看结果 =====================
print("训练完成！策略参数 θ：")
print(theta)
print("\n各状态下的动作概率：")
for s in range(n_states):
    probs = policy(s)
    print(f"状态 {s}: 左={probs[0]:.3f}, 右={probs[1]:.3f}")

print("\n走一遍最优路径：")
s = 0
path = [s]
while s != goal_state:
    probs = policy(s)
    a = np.argmax(probs)  # 选概率最大的动作
    if a == 1:
        s += 1
    else:
        s = max(0, s - 1)
    path.append(s)
print("路径:", path)
```

 核心一句话讲懂

- **Q-Learning**：学每个动作有多好（Q 值），然后选最好的
- **策略梯度**：直接学每个状态该选哪个动作的概率，用整条轨迹的回报来调整概率

---

 与 Q-Learning 的对比

| 对比项 | Q-Learning | 策略梯度 |
|--------|-----------|---------|
| 学什么 | Q 值（动作价值） | 策略 $\pi(a\|s)$ |
| 动作选择 | 贪心选最大 Q | 按概率采样 |
| 更新方式 | 单步 TD 更新 | 整条轨迹更新 |
| 适用场景 | 离散动作、小状态空间 | 连续动作、大/连续状态空间 |

### 策略学习

学习一个策略函数 $\pi(a|s)$，即给定状态 $s$ 时选择动作 $a$ 的概率。与 Q-Learning 学习状态-动作值函数不同。

下面我就用你那个 5 格迷宫的例子，手写一个**极简的策略网络**实现，只用 numpy，不用任何深度学习框架，让你一眼看懂策略网络在干嘛。

---

策略网络（Policy Network）极简实现

核心思想
- 用一个简单的神经网络（这里就是一个线性层 + softmax）来表示策略 $\pi(a|s)$
- 智能体根据策略采样动作
- 用 **REINFORCE 算法**（蒙特卡洛策略梯度）更新网络参数
- 目标是最大化累积奖励的期望

代码（可直接运行）

```python
import numpy as np

# ===================== 环境设定 =====================
n_states = 5
n_actions = 2
goal_state = 4

# ===================== 策略网络 =====================
# 用最简单的线性模型：输出 = 状态 one-hot * 权重矩阵
# 权重矩阵 shape: (n_states, n_actions)
W = np.random.randn(n_states, n_actions) * 0.1  # 小随机初始化

def policy(s):
    """给定状态 s，返回选择每个动作的概率"""
    logits = W[s]  # 取出该状态对应的权重行
    exp_logits = np.exp(logits - np.max(logits))  # 数值稳定
    probs = exp_logits / np.sum(exp_logits)
    return probs

def sample_action(s):
    """根据策略采样一个动作"""
    probs = policy(s)
    return np.random.choice([0, 1], p=probs)

# ===================== 超参数 =====================
lr = 0.01          # 学习率
gamma = 0.9        # 折扣因子
episodes = 2000    # 训练轮数

# ===================== 训练（REINFORCE） =====================
for episode in range(episodes):
    # 收集一条完整轨迹
    s = 0
    states = []
    actions = []
    rewards = []
    done = False
    
    while not done:
        a = sample_action(s)
        # 执行动作
        if a == 1:  # 右
            s_next = s + 1
        else:       # 左
            s_next = max(0, s - 1)
        
        # 奖励
        r = 10 if s_next == goal_state else 0
        done = (s_next == goal_state)
        
        states.append(s)
        actions.append(a)
        rewards.append(r)
        s = s_next
    
    # 计算折扣回报 G_t
    G = 0
    returns = []
    for r in reversed(rewards):
        G = r + gamma * G
        returns.insert(0, G)
    
    # 策略梯度更新（对每个时间步）
    for t in range(len(states)):
        s_t = states[t]
        a_t = actions[t]
        G_t = returns[t]
        
        # 计算 softmax 梯度
        probs = policy(s_t)
        # 对选中的动作 a_t，梯度 = (1 - prob[a_t]) 的负梯度形式
        # 更准确：∇log π(a|s) = one_hot(a) - probs
        grad = np.zeros(n_actions)
        grad[a_t] = 1
        grad -= probs
        
        # 更新权重：W[s_t] += lr * G_t * grad
        W[s_t] += lr * G_t * grad

# ===================== 训练完成，看结果 =====================
print("训练完成！策略网络权重 W：")
print(W)
print("\n每个状态的动作概率：")
for s in range(n_states):
    probs = policy(s)
    print(f"状态 {s}: 左={probs[0]:.3f}, 右={probs[1]:.3f}")

print("\n走一遍最优路径：")
s = 0
path = [s]
while s != goal_state:
    probs = policy(s)
    a = np.argmax(probs)  # 选概率最大的动作
    if a == 1:
        s += 1
    else:
        s = max(0, s - 1)
    path.append(s)
print("路径:", path)
```

## 核心一句话讲懂

- **策略网络**：直接学习“在某个格子应该向左还是向右”的概率
- **REINFORCE**：走完一整局，根据实际获得的奖励来调整概率——奖励高的动作概率增大，奖励低的动作概率减小
- 最终效果和 Q-Learning 一样：学会一直向右走

---

## 对比 Q-Learning vs 策略网络

| 特性 | Q-Learning | 策略网络（REINFORCE） |
|------|-----------|---------------------|
| 学习对象 | 状态-动作值函数 Q(s,a) | 策略函数 π(a|s) |
| 输出 | 每个动作的“价值” | 每个动作的“概率” |
| 动作选择 | 贪心（取最大 Q） | 采样（概率分布） |
| 适用场景 | 离散动作，状态空间小 | 连续动作，或需要随机策略 |
| 更新方式 | 单步更新（TD） | 整条轨迹更新（MC） |
