



### 强化学习

好，那我们完全不用 gym，自己手写一个最简单的迷宫游戏，只用 numpy + 随机，代码超短，一眼看懂强化学习在干嘛。
场景：
- 一共 5 个格子： [0,1,2,3,4] 
- 起点在 0，目标是走到 4
- 只能向左 / 向右走
- 走到 4 奖励 +10，游戏结束
- 其他步奖励 0
用最简单的 Q-Learning 来学。
纯手写极简强化学习代码（可直接运行）
```python
import numpy as np
import random

# ===================== 环境设定 =====================
n_states = 5  # 5个格子：0 1 2 3 4
n_actions = 2  # 0=左，1=右
goal_state = 4

# ===================== Q表初始化 =====================
Q = np.zeros((n_states, n_actions))

# ===================== 超参数 =====================
lr = 0.1  # 学习率
gamma = 0.9  # 未来奖励折扣
epsilon = 0.1  # 探索概率
episodes = 1000

# ===================== 训练 =====================
for episode in range(episodes):
    s = 0  # 每次从起点开始
    done = False
    while not done:
        # 探索 or 利用
        if random.uniform(0, 1) < epsilon:
            a = random.choice([0, 1])  # 随机走
        else:
            a = np.argmax(Q[s])  # 走Q值最大的

        # 执行动作
        if a == 1:  # 右
            s_next = s + 1
        else:  # 左
            s_next = s - 1 if s > 0 else 0

        # 奖励
        if s_next == goal_state:
            r = 10
            done = True
        else:
            r = 0

        # Q-Learning 更新公式
        Q[s, a] = Q[s, a] + lr * (r + gamma * np.max(Q[s_next]) - Q[s, a])
        s = s_next

# ===================== 训练完成，看结果 =====================
print("训练完成！Q 表：")
print(Q)
print("\n走一遍最优路径：")
s = 0
path = [s]
while s != goal_state:
    a = np.argmax(Q[s])
    if a == 1:
        s += 1
    else:
        s = max(0, s - 1)
    path.append(s)
print("路径:", path)
```

```plaintext
[ 0. 0. ]]
走一遍最优路径：
路径: [0, 1, 2, 3, 4]
```

核心一句话讲懂
- Q 表：记录每个位置“向左/向右”有多好
- 智能体一开始乱走
- 走到终点给奖励，反向更新 Q 值
- 最后学会：一直向右走
