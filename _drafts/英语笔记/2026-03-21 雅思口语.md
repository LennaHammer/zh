## 题目量与时间

总时长：11–14分钟，分3部分。
- Part 1 日常问答：题量：3–4个话题，每个3–5题，总共8–15题 时长共：4–5分钟 - 每问回答 15–30秒，2–3句话。直接回答+扩展。
- Part 2 话题卡独白：题量：1个大话题（含3–4个提示小问题） - 准备：1分钟 - 回答：1–2分钟（尽量说满2分钟），15–25句（连贯说）。
- Part 3 深度讨论：题量：5–8题（考官灵活问） - 每问：40–60秒，4–6句话 - 时长：4–5分钟

## 评分标准


雅思口语 4项评分，每项9分，取平均分。
1. Fluency & Coherence 流利与连贯 - 说话流畅，不卡顿、不停顿 - 有逻辑：先说观点，再说原因 - 会用连接词：and, but, so, because, actually...
2. Vocabulary 词汇 - 用词准确 - 有点高级词（不用太多，几个就行） - 不要一直重复简单词
3. Grammar 语法 - 句子基本没错 - 会用复杂句：if, when, because, although... - 时态别乱（过去/现在/将来别混）
4. Pronunciation 发音 - 单词读准 - 有重读、语调 - 考官能轻松听懂

最简单评分： 1. 流利不卡 2. 词汇有点高级 3. 语法少错+复杂句 4. 发音清楚

## 句型

Part 1 万能句型（2–3句就行）
- 开头 - Well, I think … - Actually, … - To be honest, …
- 表达喜好 - I really like … because … - I’m quite into … - I prefer … rather than …
- 说频率 - I usually … - I often … - Sometimes I … - I rarely …
- 直接套示例 A: Do you like reading? B: To be honest, I really like reading because it helps me relax. I usually read novels before I go to bed.
Part 2 万能结构（撑满2分钟）
- 开头 - I’d like to talk about … - The [thing/person] I want to describe is …
- 点1：介绍 - It is … - He/She is … - I’ve known him/her for …
- 点2：细节 - What I usually do is … - I remember that … - The reason why … is that …
- 点3：感受 - I feel really … - It was such a … experience - I really enjoy it because …
- 结尾 - That’s why I [like/love] it so much.
Part 3 万能句型（4–6句）
- 表达观点 - I think … - In my opinion, … - From my perspective, …
- 对比 - Compared with …, … - In the past, … but now …
- 说原因 - The main reason is that … - This is because …
- 说影响 - It has a big influence on … - It can help people to …
- 总结 - Overall, … - In general, …
