# 词缀词根


前缀

- 方向前缀：dis-=away（离开 / 分离）、ex-=out（向外 / 出）、in-=into（进入 / 向内）、re-=back（向后 / 返回）、sub-=under（在…… 之下 / 向下）
- 数量前缀：uni-=one（单一）、bi-=two（双 / 二）、tri-=three（三）、multi-=many（多）、semi-=half（半）
- 否定前缀：un-=not（不 / 非）、in-=not（不 / 无，变体 il-/im-/ir-）、dis-=not（否定 / 相反）、non-=not（非 / 不）、anti-=against（反对 / 抗）

后缀

- 名词后缀：
	- 表人 / 职业（-er/-or/-ist/-ian/-ee）；
	- 表抽象概念 / 状态（-tion/-ation/-ment/-ness/-ity/-hood）；
	- 表物（-ment/-tion/-er）
- 动词后缀：-ize/-ise（使…… 化）、-ate（使……/ 做……）、-en（使…… 变得）、-ify/-fy（使…… 具…… 性质）
- 形容词后缀：-ful（充满…… 的）、-less（无…… 的）、-able/-ible（能…… 的）、-ous/-ious（具…… 性质的）、-y（带…… 特征的）、-ive（具…… 倾向的）
- 副词后缀：-ly=way（以…… 方式）、-ward/-wards（朝…… 方向）、-wise（以…… 方式 / 方面）

词根

- 动作/器官相关： port=carry（搬运）、man/manu=hand（手）、ped=foot（脚）、struct=build（建造）
- 感知相关： vis/vid=see（看）、dict=say/speak（说）
- 自然 / 生命相关： bio=life（生命）、geo=earth（大地）
- 学科 / 研究相关： logy=study of（…… 学）
