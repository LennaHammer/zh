## 托福独立写作

替换句型

五段（开头1段+主体3段+结尾1段）。让步段也可以写在前面。

开头段

1. 背景引入：Nowadays, there is a heated debate over ___ / In contemporary society, people hold conflicting views on ___ / When it comes to ___, opinions vary widely / ___ has aroused widespread public concern
2. 双方观点：Some believe that ___, while others argue that ___ / Some maintain that ___, whereas others hold the opposite view / People are divided on whether ___ / Supporters claim ___, but critics insist ___
3. 亮明立场：Personally, I firmly agree/disagree that ___ / From my perspective, I am convinced that ___ / After careful consideration, I side with the view that ___ / I am strongly inclined to believe that ___

主体第一段

1. 论点1：To begin with, ___ / First and foremost, ___ / The primary reason is that ___ / One compelling factor is that ___
2. 解释原因：This is because ___ / The reason lies in the fact that ___ / This is mainly due to ___ / It means that ___
3. 细节展开：To be specific, ___ / More precisely, ___ / Specifically, ___ / In detail, ___
4. 举例：For example, ___ / For instance, ___ / To illustrate, ___ / A case in point is that ___
5. 段内回扣：This clearly demonstrates that ___ / This fully proves that ___ / This strongly supports my view that ___ / Obviously, this indicates that ___

主体第二段

1. 论点2：Furthermore, ___ / Moreover, ___ / In addition, ___ / Another equally important reason is that ___
2. 同义解释：In other words, ___ / Put simply, ___ / That is to say, ___ / To clarify, ___
3. 举例：For instance, ___ / To give an example, ___ / Consider ___ / A similar example is that ___
4. 论证效果：This indicates that ___ / This reflects that ___ / This makes it clear that ___ / This perfectly explains why ___
5. 小结：Therefore, it is obvious that ___ / Thus, there is no doubt that ___ / Hence, it is undeniable that ___ / Consequently, we can realize that ___

让步段

1. 承认反方：Admittedly, some people argue that ___ / It is true that ___ / Granted that ___ / It cannot be denied that ___
2. 让步认可：This view has some merits / There is some truth in this opinion / This is reasonable to some extent / This idea has certain advantages
3. 转折反驳：However, this advantage is insignificant / Nevertheless, the benefits are overshadowed by ___ / Yet this cannot change the fact that ___ / The downside outweighs this merit
4. 立场不变：Thus, my opinion still holds true / Therefore, my viewpoint remains valid / All in all, my position is still convincing

结尾段

1. 总结立场：In conclusion, I firmly believe that ___ / To sum up, I stick to my view that ___ / Taking everything into account, I insist that ___
2. 概括理由：The reasons are ___ and ___ / Not only ___, but also ___ / Considering ___ and ___
3. 收尾：For these reasons, I insist that ___ / Overall, it is clear that ___ / In short, ___ is a wise choice
