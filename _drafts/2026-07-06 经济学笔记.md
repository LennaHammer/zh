## 数据

数据类型
- 企业：财务报表
- 宏观（产业/区域）：国家统计局

财务报表
1. **利润表（损益表）**：统计一段时期营收、成本、费用，体现企业盈利亏损情况。营业利润 = 营业收入 − 成本 − 税金 − 各项费用 ± 投资收益
2. **资产负债表**：记录特定时点企业拥有的资产、背负的负债与股东权益，反映企业当下家底。公式：资产 = 负债 + 所有者权益
3. **现金流量表**：统计一段时期企业现金流入流出，展示资金真实来源与去向。期末现金 = 期初现金 + 现金净增加额
4. **所有者权益变动表**：反映当期股东权益增减变动，包含增资、分红、留存收益变化。

### 拉格朗日乘数法
 
求解**多元函数在等式约束条件下**的极值：
目标函数：$z=f(x,y)$ 约束条件：$\varphi(x,y)=0$
得到，极值的大小 f_max，和取到极值的坐标 (x,y)。

步骤：
1. 构造拉格朗日函数 $L(x,y,\lambda)=f(x,y)+\lambda\varphi(x,y)$ 	其中 $\lambda$ 称为**拉格朗日乘数** 
2. 求一阶偏导并令为0，得到方程组 $\begin{cases} L_x = f_x+\lambda\varphi_x = 0\\ L_y = f_y+\lambda\varphi_y = 0\\ L_\lambda = \varphi(x,y) = 0 \end{cases}$
3. 解方程组，得到驻点$(x,y)$；再判断是否为极值点（实际应用题一般不用严格判别）

几何理解（二元）极值点处，目标函数等高线与约束曲线**相切**，两者法向量平行：$\nabla f = -\lambda \nabla\varphi$ 梯度共线，这就是上面方程组的来源。

推广：n 元函数 + 单个约束，多个等式约束 
 

例题 ：求 $f(x,y)=xy$ 在约束 $x+y=1$ 下的极值。
1. 约束改写：$\varphi(x,y)=x+y-1=0$
2. 构造 $L=xy+\lambda(x+y-1)$
3. 求偏导： $\begin{cases} L_x=y+\lambda=0\\ L_y=x+\lambda=0\\ L_\lambda=x+y-1=0 \end{cases}$
4. 由前两式：$y=-\lambda,\;x=-\lambda \Rightarrow x=y$ 用 lambda 表示 x 和 y
5. 代入约束：$2x=1\Rightarrow x=y=\dfrac12$ 消去 lambda
6. 驻点 $\left(\dfrac12,\dfrac12\right)$，$f_{\text{max}}=\dfrac14$

注意事项
1. ❌ **不能直接处理不等式约束**（不等式约束要用KKT条件，是拉格朗日乘数法的推广）；
2. 该方法求出的只是**驻点**，不一定是极值；数学理论题需要用二阶微分（海塞矩阵）判别；
3. 使用前提：约束梯度 $\nabla\varphi \neq \boldsymbol 0$（约束曲线不出现奇点）。

KKT简要区分（不等式约束）
若约束是 $\varphi(x)\le0$，就要用KKT条件，多出互补松弛条件，是拉格朗日乘数法的升级版。

### KKT条件 
不等式约束
优化问题：$\min f(x),\; \text{s.t. } g_i(x)\le0\;(i=1,\dots,m),\;h_j(x)=0\;(j=1,\dots,l)$
最优解$x^*$满足：
1. 驻点条件：$\nabla f(x^*)+\sum_{i}\lambda_i\nabla g_i(x^*)+\sum_j\mu_j\nabla h_j(x^*)=0$
2. 原始可行性：$g_i(x^*)\le0,\;h_j(x^*)=0$
3. 互补松弛：$\lambda_i g_i(x^*)=0$
4. 对偶可行性：$\lambda_i\ge0$

通俗理解：不在边界的不等式约束$g_i<0$，对应乘子$\lambda_i=0$，不影响梯度平衡；只有激活约束$g_i=0$才参与梯度组合。
凸优化中KKT是充要条件；非凸仅为必要条件。

## 一般均衡模型

极简版一般均衡核心思想：**所有市场同时出清（供给=需求）**
1. 效用最大化：$\max u(x) \quad \text{s.t. } p\cdot x \le p\cdot \omega$ 消费者在预算约束下最大化效用，得到商品需求函数 $x(p)$。
2. 利润最大化：$\max p\cdot y$ 厂商最大化利润，得到供给函数 $y(p)$。
3. 均衡条件（瓦尔拉斯法则）：$\sum \big[x(p)-\omega-y(p)\big]=0$ 存在价格向量 $p^*$，使得每个市场超额需求等于0，全部市场同时供需平衡。
简单总结：给定价格，消费者、厂商各自最优；找到一组价格让所有市场供需相等。

## 柯布-道格拉斯函数（Cobb–Douglas）

刻画**产出**与**要素**投入的数量关系。直观描述**资本、劳动、技术**如何共同决定**总产量**，把生产过程数学化，方便定量分析，不再只做文字定性描述。


加法只能描述要素互相替代、有无对方都能生产、边际收益不变的场景；而**真实生产中资本劳动互补、缺一不可、边际产出递减、需要分解百分比增速**，只有相乘的幂函数形式才能同时满足所有经济学要求。

**生产函数最常见形式**
$$\boldsymbol{Y = A K^\alpha L^\beta}$$
符号含义：
- $Y$：**产出**
- $A$：全要素生产率（技术水平，$A>0$）
- $K$：**资本**投入
- $L$：**劳动**投入
- $\alpha$：资本产出弹性，$\alpha>0$
- $\beta$：劳动产出弹性，$\beta>0$

**2. 规模报酬判定**
1. **$\alpha+\beta=1$：规模报酬不变**（最常用！）
   $$Y=A K^\alpha L^{1-\alpha}$$
2. $\alpha+\beta>1$：规模报酬递增
3. $\alpha+\beta<1$：规模报酬递减

**3. 性质重点**
1. **对数线性**（回归常用）
两边取自然对数：
$$\ln Y=\ln A+\alpha\ln K+\beta\ln L$$
可以用线性回归估计参数。

2. **边际产出**
$$MP_K=\frac{\partial Y}{\partial K}=\alpha A K^{\alpha-1}L^\beta=\alpha\frac{Y}{K}$$
$$MP_L=\frac{\partial Y}{\partial L}=\beta A K^\alpha L^\beta=\beta\frac{Y}{L}$$

3. **要素替代弹性 $\boldsymbol{\sigma=1}$**
这是柯布-道格拉斯核心特征：资本与劳动替代弹性恒等于1。

**4. 拓展：效用函数的柯布-道格拉斯形式**
消费者效用：
$$U(x_1,x_2)=x_1^\alpha x_2^\beta$$
- x1：商品1的消费量
- x2：商品2的消费量
- 其中  和  分别反映消费者对两种商品的偏好强度（即支出份额）。
一般标准化 $\alpha+\beta=1$，参数代表支出份额。

**5. 简单例子（规模报酬不变）**
$$Y=5K^{0.3}L^{0.7}$$
资本弹性0.3，劳动弹性0.7，$\alpha+\beta=1$，规模报酬不变。

## 因果推断

**经济学因果推断：Y（结果）、X（核心处理变量）、机制变量（中介M）常用分类与经典例子**
先理清符号定义（实证因果标准记号）
- **Y：Outcome，被解释变量/结果变量（因变量）**
- **X：Treatment，核心处理/政策变量（核心自变量，我们关心其总效应）**
- **M：Mediator，机制/中介变量，X → M → Y；用来解释「X为什么会影响Y」，传导通道**
> 区分：中介M（机制通道）≠ 调节变量W（异质性，X对Y的效应大小随W变化，不是传导机制）
### X
**常见核心处理变量 X（政策/冲击/核心因果变量，经济学最常用）**
**1. 政策冲击类（双重差分DID最主流X）**
- 最低工资政策（是否实施最低工资、最低工资涨幅）
- 关税调整/贸易自由化政策
- 环保政策：排污标准、碳政策、环保督察
- 土地政策：土地出让、征地、宅基地改革
- 财税政策：增值税改革、个税改革、补贴发放、税收减免
- 社保政策：养老保险、医保扩面、失业保险
- 教育政策：义务教育、扩招、学校合并、课后服务
- 交通基建：高铁开通、高速公路、港口建设（0-1虚拟变量）
- 金融政策：信贷宽松、利率调整、普惠金融、存款保险
- 10.劳动力政策：户籍放开、居住证改革、移民政策

**2. 个体/企业特征类（工具变量IV、匹配常用）**
- 受教育年限（经典：教育回报）
- 是否参加工会、劳动合同类型
- 企业是否出口、是否研发、有无外资持股
- 数字化投入、机器人使用
- 生育（是否生二胎/三胎）
- 迁移流动（是否移民/进城务工）

**3. 市场/价格冲击（自然实验）**
- 大宗商品价格、汇率冲击
- 外生需求冲击、外生供给冲击
- 自然灾害、疫情冲击
### 结果变量 Y
**常见结果变量 Y（被解释变量）**
**1. 劳动经济学TOP高频Y**
- 工资水平、小时工资、劳动收入
- 就业状态（是否就业）、就业率、失业率
- 劳动供给：工作时长、参与劳动概率
- 劳动生产率、企业离职率、岗位创造

**2. 发展/微观家庭经济学**
- 家庭消费、储蓄、资产持有
- 子女教育成绩、入学概率、人力资本
- 生育率、婚育行为、家庭贫困状态
- 居民健康、就医行为、预期寿命

**3. 产业/企业经济学**
- 企业全要素生产率TFP：剔除资本、劳动力等有形生产要素投入增长后，由技术进步、管理优化、制度改善等带来的产出增长，衡量企业综合生产效率水平。
- 企业利润、产值、销售额
- 投资、研发投入、专利数量
- 出口额、进口中间品使用
- 污染排放、能源消耗

**4. 宏观/区域经济学（地区层面Y）**
- GDP、人均收入、经济增长率
- 城乡收入差距、地区不平等
- 房价、城市人口规模、城镇化率

**5. 公共经济学**
- 居民税负、政府财政收入
- 犯罪率、公共品供给水平
### M
**三、机制变量 M（中介变量，解释传导渠道，最经典通道分类）**
> 识别机制：检验**X → M** 以及 **控制M后，X对Y的直接效应下降**；正式检验常用中介分析、渠道检验。

按研究领域整理通用机制：
1. **（1）劳动经济学经典机制**
	1. X=教育 → Y=工资 机制M候选： 1. 生产率提升（人力资本渠道） 2. 职业选择（进入更高门槛行业） 3. 信息获取、社交网络 4. 健康水平、认知能力
	2. X=最低工资 → Y=就业 机制M： 1. 企业用工成本上升 2. 产品价格调整 3. 企业缩减雇佣/替代资本 4. 劳动力流动
2. **（2）贸易经济学（贸易自由化X→Y生产率/收入）** 常用机制M： 1. 中间投入品质量/价格 2. 竞争程度（市场竞争加剧） 3. 研发创新、技术采纳 4. 资源再配置（低效率企业退出） 5. 出口市场规模
3. **（3）基建/交通（高铁开通X→地区收入Y）** 机制M： 1. 要素流动（劳动力跨区域迁移） 2. 贸易成本降低、市场可达性 3. 产业集聚、服务业扩张 4. 信息流动、企业投资
4. **（4）环境政策（环保规制X→企业产出Y）** 机制M： 1. 治污成本上升（遵循成本效应） 2. 技术创新（波特假说渠道） 3. 要素重新分配 4. 污染转移
5. **（5）金融政策（信贷扩张X→经济增长Y）** 机制M： 1. 企业投资水平 2. 研发投入 3. 居民消费 4. 资源配置效率
6. **（6）户籍改革/移民（放开落户X→城市收入差距Y）** 机制M： 1. 劳动力供给结构 2. 行业劳动力竞争 3. 住房成本、公共服务压力

大部分经济学因果机制逃不出这六大类，你做研究可以从里面筛选：
1. **价格/成本渠道**：要素价格、生产成本、交易成本
2. **要素配置渠道**：资本、劳动力、土地跨部门流动
3. **技术/创新渠道**：研发、技术采纳、生产率
4. **市场竞争渠道**：竞争强度、进入退出、市场势力
5. **需求渠道**：市场规模、居民消费、外部需求
6. **信息与人力资本渠道**：知识扩散、技能、信息摩擦

**重要避坑提醒（因果推断机制常见误区）**
1. **中介≠调节变量**
调节W：X对Y效应大小随W变化（异质性分析），**不是传导机制**，不能叫渠道；
2. 机制变量M不能是**Y的后定变量**（时序错误：Y发生之后的变量不能作为传导通道）；
3. 简单分步回归做中介存在内生性问题；高质量论文会讨论中介内生、避免只靠逐步回归断言机制；
4. 不要混淆：X总效应 = 直接效应 + 间接效应（间接效应就是X→M→Y的机制贡献）。

## 指标构建


## 企业全要素生产率（TFP）
企业全要素生产率（TFP）是指扣除劳动、资本等有形要素投入后，由技术进步、效率提升、制度创新等无形因素带来的产出增长部分。

估算企业全要素生产率（TFP）的常见方法如下：

1. **索洛余值法（增长核算法）**  
   假设生产函数为柯布-道格拉斯形式（$Y = A K^\alpha L^\beta$），通过回归估计资本和劳动的产出弹性（$\alpha$、$\beta$），然后计算TFP增长率：  
   $\frac{\Delta A}{A} = \frac{\Delta Y}{Y} - \alpha \frac{\Delta K}{K} - \beta \frac{\Delta L}{L}$  
   其中$\frac{\Delta A}{A}$即为TFP增长率。

2. **OLS（普通最小二乘法）**  
   直接对生产函数取对数后回归：$\ln Y = \ln A + \alpha \ln K + \beta \ln L + \epsilon$，残差$\epsilon$即为TFP的对数值。

3. **固定效应法**  
   在面板数据中加入企业个体固定效应，控制不随时间变化的企业特征，减少遗漏变量偏误。

4. **OP法（Olley-Pakes，1996）**  
   用企业投资作为生产率冲击的代理变量，解决同时性偏差（企业根据当前生产率调整投入）和样本选择偏差（低生产率企业退出市场）。

5. **LP法（Levinsohn-Petrin，2003）**  
   用中间品投入（如原材料、能源消耗）替代投资作为代理变量，避免投资数据缺失或为零的问题。

6. **ACF法（Ackerberg-Caves-Frazer，2015）**  
   修正OP和LP法中劳动投入系数可能无法识别的缺陷，通过两阶段估计更稳健地得到TFP。

**实际估算步骤（以LP法为例）：**
- 收集企业面板数据：产出（通常用营业收入或增加值）、资本（固定资产净值）、劳动（员工数或工资总额）、中间投入（购买商品、劳务等）。
- 对变量取对数，设定生产函数（如Cobb-Douglas）。
- 用LP法估计资本和劳动的系数。
- 计算TFP = 实际产出 - 估计产出（对数形式），或取指数得到TFP水平值。

**注意事项：**
- 数据需平减（用价格指数将名义值转为实际值）。
- 不同方法对数据要求不同（如OP法需要投资为正，LP法需要中间投入为正）。
- 结果需结合行业特征解释，跨行业直接比较TFP可能不严谨。

## 1. 条件均值类（调整混淆）
1. **回归调整（Regression Adjustment）**：用线性/非线性回归拟合结果变量，直接控制可观测混淆变量，估计平均因果效应。
2. **倾向得分匹配（PSM）**：先用协变量算出个体接受处理的概率（倾向得分），对得分相近的处理组与对照组配对，消除协变量分布差异。
3. **倾向得分加权（IPW/逆概率加权）**：利用倾向得分构造样本权重，把观测数据重加权模拟随机试验样本，估计因果效应。
4. **双重稳健估计（DR）**：同时拟合结果回归模型与倾向得分模型，只要其中一个模型设定正确就能得到一致估计。

## 2. 断点与准实验
5. **断点回归（RDD）**：利用处理由某个连续变量阈值决定的规则，比较阈值两侧紧邻样本，识别局部平均因果效应。
6. **差分差分（DID）**：利用未受政策冲击的对照组减去自身时间趋势，消除处理组固有差异与共同时间冲击，识别政策效应。
7. **合成控制法（SCM）**：对受干预单元，用多个未受干预单元加权构造虚拟“对照个体”，对比真实与合成结果得到因果效应。

## 3. 工具变量（解决内生性/遗漏混淆）
8. **工具变量法（IV）**：寻找只通过处理变量影响结果、不直接影响结果的外生变量，剥离混杂带来的偏误识别因果效应。
9. **局部平均处理效应（LATE，IV框架）**：工具变量仅能识别服从工具诱导人群的因果效应，而非总体平均效应。

## 4. 双重选择/高维因果
10. **双重选择（Double Selection）**：分别对结果、处理变量做变量筛选，保留重要协变量，解决高维数据下遗漏变量与过拟合问题。

## 5. 图模型因果（Pearl框架）
11. **后门准则（Backdoor）**：找到阻断所有处理→结果后门路径的变量集，对该集合调整即可识别因果效应。
12. **前门准则（Frontdoor）**：当后门路径无法阻断时，借助唯一中介变量，通过中介分步识别处理对结果的因果效应。
13. **do演算（Do-Calculus）**：一套形式化运算规则，在因果图上判断干预效应能否被观测数据识别并写出识别公式。

## 6. 中介、异质性与其他
14. **中介效应分析（Mediation）**：拆分总效应为直接效应与经由中介变量传递的间接效应，揭示因果作用路径。
15. **异质性处理效应（HTE）**：估计不同特征个体因果效应差异，挖掘处理效果因人而异的规律。
16. **匹配法（粗化精确匹配CEM、精确匹配）**：直接基于协变量分层/精确配对，不依赖倾向得分函数假设平衡协变量。
17. **时间序列因果（格兰杰因果）**：基于预测能力定义，若加入过去X能提升Y预测精度，则称X是Y的格兰杰因果（注意：只是预测关联≠真实因果）。

## 7. 新兴机器学习因果
18. **因果森林（Causal Forest）**：基于随机森林自适应分割样本，灵活估计异质性处理效应，无需预设函数形式。
19. **双重/去偏机器学习（DML）**：先用机器学习拟合干扰函数，再正交化解耦处理效应，适配高维非线性混淆。
