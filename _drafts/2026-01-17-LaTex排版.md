# LaTeX的使用

## 基本用法 

以下是在建好文档后基本的文本格式操作。对于某一些非视觉的文本的效果，可以直接编辑文本也可以用一些已有的指令自动创建。

首先是输入内容，调整格式。

1.  **注释**行用 `%`

2.  **段落**用空行（两个换行）表示，一个换行会被吃掉。英文**标点**
    ``` ``引号'' ```前后是不同符号 。这样，输入一些符号需要反斜杠\
    转义。

3.  文本**样式**有 **粗体** `\textbf{}` *斜体* `\textit{}` 作为书名
    [下划线]{.underline} `\underline{}` `引文` `\verb|...|`。

4.  进入**数学公式**环境，行内使用 `$ $` 公式块使用 `\[ \]`，效果如
    $f(x)=\sin(x)$。某些环境给公式带编号用然后引用。

5.  **中文文档**使用 `\usepackage{ctex}` 并用 XeLaTeX 编译（有时要加参数
    `-shell-escape`），这样每段段首会空两格。如果仅需中文显示，只用
    `\usepackage{xeCJK}`。

6.  **多文件**项目用 `\include{...}` 作为段落 以及 `\input{...}`
    用来嵌入文本。

相比以上直接输入内容和调整格式，以下的操作的会在格式上和编号上都有自动生成的内容。

首先是会自动编号的层次型结构。

1.  **章节**用 `\section{}` 以及
    `\subsection{}`。会自动编号，用来生成对应的目录显示为 ` ` 或者 pdf
    文附带件的大纲。

2.  **列表** 用 `\begin{itemize}` 编号列表（会自动按前后顺序编号）用
    `\begin{enumerate}` ，其中的列表项目用 `\item` 。

然后是编号的前后引用。

为了在当前论文的不同位置或元素之间，以及相关文档之间建立链接，并由系统自动维护编号，可以使用交叉引用和参考文献功能。

1.  **交叉引用** 使用
    `\ref{label:1}`（显示为数字编号），用来引用章节、图片（以及页码、公式）。在被引用对象后面添加
    `\label{label:1}`。例如当前是在第
    [\[label1\]](#label1){reference-type="ref" reference="label1"}
    节、第 页。

2.  **参考文献**用BibTeX 把文献列表放在 .bib 文件中，然后 `\cite{...}`
    引用.需要文档最后放上
    `\bibliography{bib文件名}`，并多编译几次，就不显示为问号了。这里显示为[@einstein]。通常习惯取名为作者+年份+短标题。

对于论文中出现的非文本元素，如图表、表格和伪代码，以如下方式插入。这些元素可以出现在文本中，也通常浮动float在论文页面的固定位置。

1.  插入**图片**用 `\includegraphics{}`。通常放在 `\begin{figure}`
    通常位置跟随页面在顶端*居中*并包含*标题*
    `\caption{...}`，不随跟随文本位置，然后通过交叉引用来指明。

2.  **表格**（含线） `\begin{tabular}{|c|c|c|}\hline` 用 `&` 和 `\\`
    分隔列和行。线条样式常用三线表。表格位置可以类似图片放在
    `\begin{table}` 中跟随页面，并通过引用来使用。

3.  **代码块** 用 `\begin{verbatim}` 行内用
    `\verb||`，如需伪代码或代码高亮可以用其他的包。通常位置类似图片。

编辑tex文件，直接使用任何常用的文本编辑器都是可以的。一些综合的原文件的例子，见这个文档的后面的内容。编译文档到PDF文件可以使用命令行，或者选择让IDE自动进行。

规划求解（需要在插件中开启）

每个变量占一个相邻的单元格，目标函数=公式的形式写在单元格了，然后在求解器对话框中执行是哪些单元格。
