## 产业经济学

产业经济学的分析维度可从**核心框架（SCP）**与**五大研究领域**两个层面把握，核心是**产业组织、产业结构、产业布局、产业发展、产业政策**五大维度。 ### 一、


核心分析框架：SCP范式（最基础） 
- **市场结构（Structure）**：行业集中度、企业数量、规模分布、进入/退出壁垒、产品差异化。 
- **市场行为（Conduct）**：定价策略、广告与研发、并购与联盟、竞争/合谋行为。 
- **市场绩效（Performance）**：资源配置效率、利润率、技术进步、消费者福利、规模效率。 ### 二、


五大核心研究维度（宽派主流） 1. **产业组织（中观-企业间）**：SCP范式，研究同一产业内企业竞争与垄断关系。 2. **产业结构（产业间）**：三大产业比例、产业关联（投入产出）、结构演进与升级规律。 3. **产业布局（空间维度）**：产业区域分布、集聚效应、区位选择、区域协调发展。 4. **产业发展（时间维度）**：生命周期、成长动力、技术创新、全球化与产业链演变。 5. **产业政策（政府维度）**：政策制定、实施效果、规制与反垄断、产业扶持与转型政策。 ### 三、

补充视角（常用延伸） - **产业链/价值链**：上中下游协同、价值分配、供应链安全。 - **技术与创新**：技术链、创新链、技术溢出与产业升级。 - **国际维度**：全球价值链、国际竞争与贸易、跨国产业转移。 要不要我把这些维度整理成一份可直接套用的分析清单（含关键指标与常用模型），方便你快速上手？


### 产业经济学指标和计算方法


**SCP（结构-行为-绩效）范式** 来优化您提供的“产业经济学常用指标”列表，是一个非常棒的思路。SCP范式强调市场结构（Structure）决定企业行为（Conduct），进而影响市场绩效（Performance）。产业经济学常用指标

一、市场结构（Structure）

衡量产业内竞争与垄断程度、进入壁垒、产品差异化等。

- **规模类**
	- **产业增加值**：总产出 - 中间消耗。衡量产业对经济的绝对贡献。
	- **全员劳动生产率**：产业增加值 / 从业人数。衡量单位劳动力的产出效率。

- **集中度与垄断程度**
	- **产业集中度CRn**：$CR_n = \frac{前n家企业营收总和}{行业总营收} \times 100\%$。衡量市场垄断或竞争程度。
	- **赫芬达尔HHI指数**：$HHI = \sum_{i=1}^n (市场份额_i)^2$（份额用百分数数值计算）。综合衡量市场集中度，数值越高说明垄断程度越高。

- **专业化与集聚**
	- **区位熵**：$LQ = \frac{地区某产业产值 / 地区总产值}{全国该产业产值 / 全国总产值}$。判断某产业在地区的专业化与集聚水平。

- **工业化阶段**
	- **霍夫曼系数**：消费品工业产值 / 资本品工业产值。反映工业化阶段（重工业化程度）。

- **进入壁垒与产品差异化**
	- **最小有效规模（MES）**：长期平均成本最低时的最小产量。衡量规模经济壁垒。
	- **广告密度**：广告支出 / 销售收入。反映产品差异化程度和品牌壁垒。
	- **研发投入强度**：R&D经费支出 / 产业增加值 × 100%。衡量创新壁垒和技术门槛。

二、市场行为（Conduct）

衡量企业在定价、研发、广告、并购等方面的策略与行为。

- **定价行为**
	- **勒纳指数（Lerner Index）**：$L = \frac{P - MC}{P}$。衡量企业定价权与垄断利润能力。
	- **成本费用利润率**：利润总额 / 成本费用总额 × 100%。反映成本控制与定价策略效果。

- **创新行为**
	- **研发投入强度**：R&D经费支出 / 产业增加值 × 100%。衡量创新意愿和资源投入。
	- **专利密度**：专利申请数 / 从业人数（或产业增加值）。衡量创新产出密度。
	- **新产品产值率**：新产品产值 / 产业总产值 × 100%。衡量创新成果的商业化转化能力。

- **竞争与合作行为**
	- **并购数量与金额**：反映产业整合与竞争格局变化。
	- **战略联盟数量**：反映企业间合作与协同行为。

三、市场绩效（Performance）

衡量产业在资源配置效率、技术进步、利润水平、社会福利等方面的最终结果。

- **效益类**
	- **产值利税率**：(利润总额 + 税金) / 产业总产值 × 100%。衡量单位产出的利税贡献。
	- **净资产收益率ROE**：净利润 / 平均净资产 × 100%。衡量资本回报效率。
	- **贝恩指数**：(会计利润 - 正常利润) / 权益资本。衡量超额利润水平。
	- **托宾Q值**：$Q = \frac{企业市场价值}{企业重置成本}$。反映市场对企业未来盈利的预期和投资价值。

- **效率与增长核算类**
	- **全要素生产率（TFP）**：$TFP = \frac{Y}{K^\alpha L^{1-\alpha}}$（索洛余值法），或通过 DEA、SFA 等方法测算。衡量技术进步、管理效率等综合贡献。
	- **技术进步率**：产出增长率 - α × 资本增长率 - (1-α) × 劳动增长率。分解增长来源。
	- **资本产出比**：资本存量 / 产业增加值。衡量资本利用效率。
	- **劳动产出弹性**：(ΔY/Y) / (ΔL/L)。反映劳动投入对产出的敏感度。

- **增长与产能类**
	- **产业增长率**：(本期产值 - 上期产值) / 上期产值 × 100%。衡量产业发展速度。
	- **产能利用率**：实际产量 / 设计产能 × 100%。判断是否存在产能过剩或供给不足。

- **产业关联类**
	- **影响力系数**：某产业完全消耗系数均值 / 全部产业完全消耗系数总均值。衡量某产业对国民经济的拉动作用（前向关联）。
	- **感应度系数**：某产业完全分配系数均值 / 全部产业完全分配系数总均值。衡量某产业受国民经济各产业需求拉动的影响程度（后向关联）。
	- **中间投入率**：中间投入总额 / 产业总产出。反映产业对上游的依赖程度。

- **开放与能耗类**
	- **外贸依存度**：进出口总额 / 地区GDP × 100%。衡量产业与国际市场的联系强度。
	- **单位产值能耗**：能源消耗总量 / 产业增加值。衡量产业的绿色化水平和能源利用效率。

- **产业安全与风险类**
	- **产业对外依存度**：关键零部件进口额 / 产业总产值 × 100%。衡量关键环节的供应链风险。
	- **产业控制力指数**：内资企业产值 / 产业总产值 × 100%。衡量本土资本对产业的主导权。

## 演示脚本
###  上市公司财报分析
2. **上市公司财务报表分析**（用免费开源库 `AkShare` 获取真实财务数据，做基础指标计算+可视化）
核心功能
获取上市公司利润表 → 清洗数据 → 计算**毛利率、净利率**核心指标 → 营收/净利润可视化
依赖库
`akshare`（免费获取财务数据）、`pandas`（数据处理）、`matplotlib`（画图）
```bash
# 安装依赖
pip install akshare pandas matplotlib
```

完整演示代码
```python
# 1. 导入库
import akshare as ak
import pandas as pd
import matplotlib.pyplot as plt

# 解决中文显示问题
plt.rcParams["font.family"] = ["SimHei", "WenQuanYi Micro Hei", "Heiti TC"]
plt.rcParams["axes.unicode_minus"] = False

# ---------------------- 2. 获取上市公司财务数据（贵州茅台：600519） ----------------------
# 获取利润表（新浪财经接口，免费无token）
finance_df = ak.stock_financial_report_sina(symbol="600519", report_type="利润表")

# 筛选最新5期数据（简化分析）
df = finance_df.head(5).copy()
# 重命名列（方便理解）
df = df.rename(columns={"报告期": "report_date", "营业收入": "revenue", "营业成本": "cost", "净利润": "profit"})

# ---------------------- 3. 核心财务指标计算 ----------------------
# 毛利率 = (营业收入-营业成本)/营业收入 *100%
df["gross_margin"] = (df["revenue"] - df["cost"]) / df["revenue"] * 100
# 净利率 = 净利润/营业收入 *100%
df["net_margin"] = df["profit"] / df["revenue"] * 100

# ---------------------- 4. 输出分析结果 ----------------------
print("="*50)
print("贵州茅台(600519) 核心财务指标分析")
print("="*50)
print(df[["report_date", "revenue", "profit", "gross_margin", "net_margin"]])

# ---------------------- 5. 可视化：营业收入 vs 净利润 ----------------------
plt.figure(figsize=(10, 5))
x = df["report_date"]
plt.bar(x, df["revenue"]/10000, label="营业收入(万元)", alpha=0.7)
plt.bar(x, df["profit"]/10000, label="净利润(万元)", alpha=0.9)
plt.title("贵州茅台 营业收入&净利润趋势")
plt.ylabel("金额(万元)")
plt.legend()
plt.xticks(rotation=30)
plt.tight_layout()
plt.show()
```

演示效果
1. 控制台打印**报告期、营收、净利润、毛利率、净利率**；
2. 弹出柱状图，直观对比营业收入和净利润；
3. 毛利率≈90%、净利率≈50%，符合茅台真实财务特征。

### 政策文本分析

2. **政策文本分析**（用中文NLP库做分词、词频、词云、情感分析）
核心功能
模拟政策文本 → 中文分词 → 词频统计 → 生成词云 → 情感倾向分析
依赖库
`jieba`（中文分词）、`wordcloud`（词云）、`snownlp`（情感分析）
```bash
# 安装依赖
pip install jieba wordcloud snownlp pandas
```

完整演示代码
```python
# 1. 导入库
import jieba
from wordcloud import WordCloud
import matplotlib.pyplot as plt
from snownlp import SnowNLP
import pandas as pd

# 解决中文显示
plt.rcParams["font.family"] = ["SimHei", "WenQuanYi Micro Hei", "Heiti TC"]
plt.rcParams["axes.unicode_minus"] = False

# ---------------------- 2. 模拟政策文本（数字经济发展政策） ----------------------
policy_text = """
为推动数字经济高质量发展，强化科技创新驱动，加快数字基础设施建设，
培育数据要素市场，支持中小企业数字化转型，加强网络安全保障，
完善数字经济治理体系，促进数字技术与实体经济深度融合。
"""

# ---------------------- 3. 中文分词 + 停用词过滤 ----------------------
# 自定义停用词（无意义虚词）
stop_words = {"的", "和", "在", "要", "为", "是", "与", "等", "加强", "加快", "推动"}
# 分词
words = jieba.lcut(policy_text.strip())
# 过滤停用词和空字符
filtered_words = [w for w in words if w not in stop_words and len(w) > 1]

# ---------------------- 4. 词频统计 ----------------------
word_count = pd.Series(filtered_words).value_counts().head(10)
print("="*50)
print("政策文本高频词TOP10")
print("="*50)
print(word_count)

# ---------------------- 5. 生成词云图 ----------------------
wc = WordCloud(
    font_path="simhei.ttf",  # 中文字体
    background_color="white",
    width=800,
    height=400
).generate(" ".join(filtered_words))

# ---------------------- 6. 情感分析（判断政策积极/消极） ----------------------
s = SnowNLP(policy_text)
sentiment_score = s.sentiments  # 0~1，越接近1越积极
sentiment = "积极" if sentiment_score > 0.5 else "消极"

print("\n" + "="*50)
print(f"政策文本情感倾向：{sentiment}")
print(f"情感得分：{sentiment_score:.2f}")
print("="*50)

# 显示词云
plt.figure(figsize=(10, 5))
plt.imshow(wc)
plt.axis("off")
plt.title("政策文本词云图")
plt.show()
```

演示效果
1. 输出政策文本**高频词**（数字经济、科技创新、实体经济等）；
2. 弹出**词云图**，核心政策关键词一目了然；
3. 输出**情感分析结果**：政策文本为**积极倾向**（得分≈0.99）。

### akshare

AKShare 是开源财经数据接口库，覆盖**股票、基金、期货、期权、债券、外汇、宏观、指数、加密货币**等全品类金融数据，接口超 2000 个，核心模块如下：

- 一、股票数据（A股/港股/美股） - **实时行情**：盘口、分时、逐笔成交 - **历史K线**：日/周/月/分钟线、复权数据 - **财务数据**：资产负债表、利润表、现金流量表、业绩预告 - **资金流向**：主力资金、龙虎榜、北向/南向资金 - **股东与分红**：十大股东、流通股东、分红送转、限售解禁 - **板块与指数**：行业/概念板块行情、成分股、指数估值
- 二、基金数据 - **净值与收益**：单位净值、累计净值、收益率、回撤 - **持仓与评级**：股票/债券持仓、基金经理、晨星/天相评级 - **规模与发行**：基金规模、成立时间、申购赎回费率
- 三、期货与期权数据 - **期货**：商品/金融期货（股指、国债）、实时行情、历史K线、持仓量、基差、仓单 - **期权**：50ETF/沪深300ETF期权、隐含波动率、行权价链、 Greeks、期权链数据
- 四、债券数据 - **利率与曲线**：中债收益率曲线、回购定盘利率、同业拆借利率 - **交易与发行**：现券交易、回购数据、发行公告、信用评级、违约风险
- 五、外汇与加密货币 - **外汇**：人民币汇率、美元指数、主要货币对实时/历史数据、外汇储备 - **加密货币**：比特币/以太坊等币价、K线、成交量、CME比特币期货

宏观经济数据
- **国内**：GDP、CPI、PPI、PMI、工业增加值、社融、货币供应量、进出口
- **国际**：美国非农、美联储利率、欧洲CPI、全球主要经济体指标
- **行业与政策**：行业景气度、财政/货币政策、进出口数据

指数与另类数据
- **全球指数**：道琼斯、纳斯达克、标普500、富时罗素、恒生指数
- **另类数据**：电影票房、电商销量、舆情数据、胡润/福布斯富豪榜

以下是一个简单的策略示例：

```python
# pip install akquant
import akquant as aq
import akshare as ak
from akquant import Strategy

# 1. 准备数据
# 使用 akshare 获取 A 股历史数据 (需安装: pip install akshare)
df = ak.stock_zh_a_daily(symbol="sh600000", start_date="20250212", end_date="20260212")


class MyStrategy(Strategy):
    def on_bar(self, bar):
        # 简单策略示例:
        # 当收盘价 > 开盘价 (阳线) -> 买入
        # 当收盘价 < 开盘价 (阴线) -> 卖出

        # 获取当前持仓
        current_pos = self.get_position(bar.symbol)

        if current_pos == 0 and bar.close > bar.open:
            self.buy(symbol=bar.symbol, quantity=100)
            print(f"[{bar.timestamp_iso}] Buy 100 at {bar.close:.2f}")  # UTC ISO 8601

        elif current_pos > 0 and bar.close < bar.open:
            self.close_position(symbol=bar.symbol)
            print(f"[{bar.timestamp_iso}] Sell 100 at {bar.close:.2f}")  # UTC ISO 8601


# 运行回测
result = aq.run_backtest(
    data=df,
    strategy=MyStrategy,
    initial_cash=100000.0,
    symbols="sh600000"
)

# 打印回测结果
print("\n=== Backtest Result ===")
print(result)

# 生成最小基准对比报告
benchmark_returns = (
    df.set_index("date")["close"].pct_change().fillna(0.0).rename("SIMPLE_BENCH")
)
result.report(
    filename="quickstart_report.html",
    show=False,
    benchmark=benchmark_returns,
)
```

调用 `result.report(..., benchmark=...)` 后，报告会新增“基准对比 (Benchmark Comparison)”区块，展示策略/基准/超额累计收益曲线，以及累计超额收益、年化超额收益、跟踪误差、信息比率、Beta、Alpha 等相对指标。

```python
# -*- coding: utf-8 -*-
import time
import akshare as ak
from snownlp import SnowNLP
 
# 使用snownlp
stock_code = '603777'
date = time.strftime("%Y%m%d", time.localtime())
stock_news_em_df = ak.stock_news_em(stock=stock_code)
 
for i in stock_news_em_df.values[:, 1]:
    text=str(i)
    # text = u'中国人是好人'
    s = SnowNLP(text)
    for sentence in s.sentences:
        print(sentence, SnowNLP(sentence).sentences)
    print(s.sentiments)
    print(s.keywords(3))
    print(s.summary(3))
    # 小于0.4的为消极，否则为积极
    if s.sentiments<0.4:
        print('##########消极',i)
    elif s.sentiments>=0.4:
        print('##########积极',i)
```
